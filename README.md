# Popcorn Irregular Verbs Game

Upload everything in this folder to a GitHub repository.

For GitHub Pages:

1. The file `index.html` must stay in the main/root folder of the repository.
2. Go to the repository Settings.
3. Open Pages.
4. Choose Deploy from a branch.
5. Select `main` and `/root`.
6. Save.

The activity will open from the GitHub Pages link.
